Install:
(1) download the tar package
(2) tar -xvf fault-tolerant-distributed-system.tar
(3) get in to fault-toerant-distributed-system directory
(4) change the 4 IPs in servers.sh to the 4 servers IP. *Do not change the ports in the 2nd line*
(5) ./deploy.sh



(*) Before using the system, we need to make sure the time of the servers are synchronized
(*) The ip address should not change during the reboot
(*) The port number of different servers in config.txt should be different
